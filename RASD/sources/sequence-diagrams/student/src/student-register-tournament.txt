title Student registers in a tournament

activate Logged student
activate CKB

Logged student->CKB:get student dashboard
CKB-->Logged student:student dashboard page
Logged student->CKB:search for tournament
alt existing tournament
Logged student->CKB:click "More information on this tournament!" button
CKB-->Logged student:tournament information page
Logged student->CKB: click "Join tournament!" button
alt deadline not passed yet
CKB-->Logged student:registration succesfull
else deadline has passed
CKB-->Logged student:deadline has passed error
end
else nonexistent tournament
CKB-->Logged student:nonexistent tournament error
end
